# TursMovie-app-
TursMovie adalah aplikasi manajemen bioskop yang mendukung operasional bioskop skala kecil hingga menengah. Sistem ini menyediakan dua mode pemesanan (online dan offline/walk-in) yang terintegrasi, manajemen jadwal film, pengelolaan pesanan makanan dan minuman, serta pengelolaan ruangan dan perangkat pemutaran film.
